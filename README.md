# demoAPI : Restaurant Mangement

## APIs 

### User

- /user/register
- /user//{userID}/placeOrder
- /user/{userID}/addReview
- /user/{id}/search
- /user/{userID}/order/{orderID}/food/{foodID}

### Restaurant

- /restaurant/register
- /restaurant/update
- /restaurant/{restaurantID}/food/{foodID}/order/{orderID}

### Food

- /food/add
- /food/getFoods
- /food/remove
- /food/deactivate
